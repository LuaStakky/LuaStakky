# LuaStakky
LuaStakky - full stack lua web framework

## What's under the hood?
+ Docker (docker-compose)
+ Openresty (Nginx with lua) with build-in modules:
    * lua-cmsgpack
    * etlua (lua-based template language)
    * inspect
    * lua-resty-http
    * lua-resty-cookie
+ Tarantool (DB)
+ fengari (js lua implementation)
